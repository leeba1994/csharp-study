## 📝 학생 특성 및 학습 스타일 (Claude 참고용)

### 기본 특성
- **CCTV 유지보수 업무 최근 시작한 초보자** (경력자 아님)
  → RTSP, ONVIF, NVR 등 CCTV 용어는 친절히 풀어서 설명해야 함
- **영어 어려워함** → 모든 설명 한국어, 영어 자료 제시 시 한국어 해설 필수
- **프로그래밍 입문자** (학원 이력 있으나 기억 가물가물)

### 학습 스타일 (지금까지 관찰한 것)
- ✅ **호기심이 많음** → "왜 이거 써?", "더 좋은 거 없어?" 같은 좋은 질문 자주 함
- ✅ **직관이 정확함** → C# WPF가 왜 필요한지 본능적으로 맞춤
- ✅ **스스로 판단함** → Claude 제안에 "이건 아닌 것 같다"고 반박할 수 있음 (건강한 태도)
- ✅ **실용적 사고** → "참고용 노트니까 깔끔해야 한다" 같은 주도적 제안 가능
- ⚠️ **본인 말로 정리는 아직 어색함** → Claude가 정리한 내용을 그대로 복사하려는 경향 (본인 학습 스타일로 존중)

### 선호하는 설명 방식
- **짧게보다 충분히** 설명하는 것을 선호
- **비유를 좋아함** (요리사, CCTV 비유 등)
- **표와 계층 구조 다이어그램**을 좋아함
- **나중에 참고할 노트**는 깔끔하게 정리되어 있는 걸 선호

### 주의할 점
- **본인 말로 쓰라고 강요하지 말 것** (학생이 이미 명확히 선호 밝힘)
- **Phase 벗어난 기술 섞지 말 것** (Phase 1 중엔 C#에만 집중)
- **중요한 결정은 반드시 학생에게 선택권 줄 것**
